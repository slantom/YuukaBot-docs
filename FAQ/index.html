# 为什么在 QQ 下私聊 Bot 没有反应

由于在 QQ 平台下的 不可抗力 缘故，目前进行私聊回复极易造成风控乃至封号，故暂不开放私聊服务。

# 为什么在 QQ 下 Bot 会突然消失一段时间

由于腾讯的一些 不可抗力 ，Bot 可能经常被风控或封号，请不要担心。
